This folder contains the files needed to use the KH2 VGMTrans export batch.

Main entry point:
  VGMTransExportKh2.bat

It uses:
  vgmtrans-cli.exe

Typical usage:
  Drag one or more .wd / .bgm files onto VGMTransExportKh2.bat

Notes:
  - VGMTrans exports reconstructed MID/SF2/DLS data, not exact in-game playback.
  - For KH2 testing, compare structure and mapping, not only sound.
